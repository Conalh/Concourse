## [0.1.0] - 2026-07-16

### Added

- A local-first React learning suite with Today, Routes, Loop, Transfer,
  Progress, profile, and settings flows.
- A Tauri desktop shell with a user-selected Courses folder and native,
  durable installed-pack records.
- Framework-independent learning-pack contracts plus Node and browser SDK
  entry points for validation, deterministic archives, canonical hashing,
  release diffs, and atomic installs.
- A CC BY 4.0 Logic Foundations course and distributable `.learntpack` archive.
- Defensive archive limits, canonical-byte verification, and atomic native
  persistence for untrusted pack input.
- Cross-platform CI, contributor guidance, security reporting, and an MIT
  software license.

### Changed

- Replaced the named development learner configuration with fictional demo
  data.
- Aligned package, browser, desktop, and public documentation under the
  Concourse v0.1 release identity.
