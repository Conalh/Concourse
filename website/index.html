04 / Project truth

Built enough to use. Early enough to shape.

Delivered in v0.1

  • Browser and Tauri desktop learning surfaces
  • Deterministic sessions, evidence, feedback, and recaps
  • Portable pack contracts, validation, archives, and SDK tooling
  • Local persistence and defensive pack installation
  • An openly licensed Logic Foundations example pack

The local-first boundary

  • No accounts
  • No hosted learner state
  • No analytics
  • No application backend
  • Progress and installed packs stay local
